# 😎 Moodify – Your Daily Mood Selector App

A simple and aesthetic Flutter app that lets users express and track their daily mood with a clean UI and smooth interactions.

---

## 🚀 Features

- 🎭 Select your mood (Happy, Sad, Angry, Chill)
- 🎨 Dynamic UI (background & message change based on mood)
- 📜 Mood History (ListView)
- 🔘 Multiple Button Types (Elevated, Outlined, Icon, Text)
- 🪄 Snackbar feedback on mood update
- 🧹 Clear history option
- 🌈 Gradient background for aesthetic look

---

## 🛠️ Tech Stack

- Flutter
- Dart

## ▶️ How to Run

1. Clone the repository:
  """ git clone https://github.com/detective-ved/moodify.git """

2. Navigate to project:
     """ cd moodify """
   
3. Get dependencies:
   """ flutter pub get """
   
4. Run the app:
   """ flutter run """
